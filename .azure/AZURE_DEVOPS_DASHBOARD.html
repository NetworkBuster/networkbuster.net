üöÄ Deployment Guide
Quick Deploy: Follow these steps to deploy to Azure Container Apps

Prerequisites

  • Azure CLI: winget install microsoft.azurecli
  • Docker: winget install docker.dockerdesktop
  • Azure Account: Create Free Account

Deployment Steps

1. Login to Azure
az login
2. Create Resource Group
az group create --name networkbuster-rg --location eastus
3. Create Container Registry
az acr create --resource-group networkbuster-rg --name networkbusteracr --sku Basic
4. Build & Push Docker Image
az acr build --registry networkbusteracr --image networkbuster:latest .
5. Create Container App Environment
az containerapp env create --name networkbuster-env --resource-group networkbuster-rg --location eastus
6. Deploy Container App
az containerapp create --name networkbuster --resource-group networkbuster-rg --environment networkbuster-env --image networkbusteracr.azurecr.io/networkbuster:latest --target-port 3000 --ingress external
