Branded assets for the NetworkBuster installer.

Place your final assets here before building the installer:
- `logo.svg` — The main project logo (SVG preferred).
- `banner.png` — A 640x120 installer banner (PNG).
- `header.png` — A 300x80 header for installer pages (PNG).
- `icon.ico` — Optional ICO (if present will be embedded into shortcuts and installer).

Use `convert-icon.ps1` to generate an `icon.ico` from `icon-placeholder.png` if needed (requires ImageMagick).
